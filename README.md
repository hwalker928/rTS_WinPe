# WinPE

#### Use

You need the Windows PE ADK installed.

https://docs.microsoft.com/en-us/windows-hardware/manufacture/desktop/download-winpe--windows-pe

Run `build.ps1`

#### Added modules
* WinPE-WMI
* WinPE-NetFX
* WinPE-Scripting
* WinPE-PowerShell
* WinPE-SecureBootCmdlets
* WinPE-SecureStartup
* WinPE-StorageWMI
* WinPE-DismCmdlets
* WinPE-EnhancedStorage
* WinPE-FMAPI
* WinPE-WiFi-Package
* WinPE-PlatformID
